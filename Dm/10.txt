The text in the image is as follows:

    1. Configure Replica set
    2. Start Mongodb in the SERVICES
    3. Create separate data directions for each replica set member in "C:\data\db1", "C:\data\db2", "C:\data\db3"
    4. Edit the mongod.cfg file present in MongoDB Folder
    5. Locate the replication section and add the replSetName parameter. choose a unique name for your replica set replication:
    - replSetName: "myReplicaSet"
    5. Open a separate command prompt for each replica set member
    6. Start each mongod instance, specifying the replica set name, data path, port and bind-ip:
    - mongod --port 2717 --dbpath "C:\data\db1" --replSet myReplicaSet --bind_ip localhost
    - mongod --port 2717 --dbpath "C:\data\db2" --replSet myReplicaSet --bind_ip localhost
    - mongod --port 2717 --dbpath "C:\data\db3" --replSet myReplicaSet --bind_ip localhost
